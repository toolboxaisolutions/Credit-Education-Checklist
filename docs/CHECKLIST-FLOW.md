# Checklist Flow — The Complete User Journey

## Overview

16 tasks across 5 phases. Sequential gates with some parallel tracks. Target completion: 6-9 months.

---

## Phase 1: Foundation (Week 1-2)
**Theme:** Know where you stand.

```
Task 1 ─┬─ Task 2 (parallel)
        │
        └─ Task 3 ─── Task 4
```

---

### Task 1: Pull Your Free Credit Reports
- **Time:** 15 min
- **Gate:** Must complete before Phase 2 unlocks
- **Status:** Required

**User sees:**
> Your credit report is the raw data — the full picture of every account, every late payment, every error. You can't fix what you can't see.

**Instructions:**
1. Go to AnnualCreditReport.com (the ONLY free official source)
2. Pull from all three bureaus: Experian, Equifax, TransUnion
3. Download/save each report
4. Scan for red flags — highlight anything suspicious

**Sub-checklist:**
- [ ] Pulled Experian report
- [ ] Pulled Equifax report
- [ ] Pulled TransUnion report
- [ ] Saved all three reports
- [ ] Highlighted any errors or suspicious accounts

---

### Task 2: Understand Your Credit Score ⚡ PARALLEL
- **Time:** 10 min
- **Gate:** None (can do alongside Task 1)
- **Status:** Optional but recommended

**User sees:**
> Your credit score is a 3-digit number (300-850) that summarizes your credit health. Here's what drives it — and how to move it up.

**Instructions:**
1. Learn the 5 factors that make up your FICO score:
   - Payment history (35%)
   - Credit utilization (30%)
   - Length of credit history (15%)
   - Credit mix (10%)
   - New inquiries (10%)
2. Pull your free score from Credit Karma, Experian, or your bank's app
3. Note your starting score — we'll track it over time

**Sub-checklist:**
- [ ] Read about the 5 factors
- [ ] Pulled current credit score
- [ ] Entered score in tracker (optional)

---

### Task 3: Set Up Your Budget
- **Time:** 30 min
- **Gate:** Must complete Task 1
- **Status:** Required

**User sees:**
> Before you start fixing your credit, you need to know what you can afford to pay each month. This takes the guesswork out.

**Instructions:**
1. Download the provided budget template (Google Sheets/Excel)
2. List all income sources
3. List all monthly expenses
4. Calculate disposable income
5. Determine how much you can put toward debt payments

**Sub-checklist:**
- [ ] Downloaded budget template
- [ ] Listed all income
- [ ] Listed all expenses
- [ ] Calculated disposable income
- [ ] Set target debt payment amount

---

### Task 4: Build a Safety Net
- **Time:** 15 min
- **Gate:** Must complete after Task 3
- **Status:** Required

**User sees:**
> One emergency = new debt. A small savings cushion keeps you from falling backward while you rebuild.

**Instructions:**
1. Set target: $500–$1,000 emergency fund
2. If you don't have one, open a high-yield savings account
3. Set up automatic transfer ($25-50/week to start)
4. Recommended: Ally, Marcus, Discover Savings

**Sub-checklist:**
- [ ] Set emergency fund goal
- [ ] Opened savings account (or identified existing one)
- [ ] Set up automatic monthly transfer

**🏆 PHASE 1 COMPLETE:** "You now know where you stand, what you can spend, and you have a safety net. Now we clean up the past."

---

### ⛔ CRITICAL GATE: Must complete all Phase 1 tasks before Phase 2 unlocks.

## Phase 2: Clean Up (Week 2-6)
**Theme:** Fix what's broken.

```
Task 5 (Disputes) ─┐
                    ├─ Task 7 (Autopay)
Task 6 (Collections) ┘
```

---

### Task 5: Dispute Inaccuracies
- **Time:** 45 min (per bureau)
- **Gate:** Must complete Phase 1. Must complete before Phase 3 unlocks.
- **Status:** Required

**User sees:**
> 1 in 5 credit reports have errors. If there's a late payment you actually paid on time, an account that isn't yours, or a collection already resolved — you can get it removed. This is FREE and LEGAL.

**Instructions:**
1. Review your highlighted items from Task 1
2. Categorize: "Clearly wrong" vs. "Maybe wrong"
3. For each clearly wrong item, file a dispute with the relevant bureau
4. Download dispute letter templates provided
5. File online when possible (faster, you can attach documents)

**Sub-checklist:**
- [ ] Identified all errors from credit reports
- [ ] Filed dispute with Experian
- [ ] Filed dispute with Equifax
- [ ] Filed dispute with TransUnion
- [ ] Saved confirmation/reference numbers
- [ ] Set 30-day follow-up reminder

---

### Task 6: Handle Collections ⚡ PARALLEL
- **Time:** 30 min per account
- **Gate:** Must complete Phase 1. Can run parallel with Task 5.
- **Status:** Required if collections exist; skip if none

**User sees:**
> Collections accounts drag your score down for up to 7 years. But you have options — and some are better than others.

**Instructions:**
1. List all collection accounts
2. For each one, decide: dispute, negotiate, or pay-for-delete
3. Use the provided call scripts
4. NEVER give a collection agency your bank info directly
5. Get all agreements IN WRITING before paying anything

**Sub-checklist:**
- [ ] Listed all collection accounts
- [ ] Decided strategy for each (dispute / negotiate / pay-for-delete)
- [ ] Contacted each collector
- [ ] Documented all conversations
- [ ] Saved written agreements

---

### Task 7: Set Up Autopay
- **Time:** 20 min per account
- **Gate:** Must complete Task 5
- **Status:** Required

**User sees:**
> One late payment wipes out months of progress. Autopay makes sure it NEVER happens again.

**Instructions:**
1. List all current accounts with minimum payments
2. For each, set up automatic payment from your bank
3. Set for the due date OR a few days before
4. Keep manual payment as backup (set phone reminder for due date - 3 days)

**Sub-checklist:**
- [ ] Listed all active accounts
- [ ] Set autopy for each account
- [ ] Set phone backup reminders
- [ ] Confirmed bank account has sufficient funds

**🏆 PHASE 2 COMPLETE:** "Errors disputed. Collections handled. Future late payments blocked. You've done the hardest part. Now we build."

---

### ⛔ CRITICAL GATE: Must complete Tasks 5 AND 7. Task 6 is parallel but not a blocker.

### Task 8: Apply for a Secured Credit Card
- **Time:** 20 min
- **Gate:** Must complete Phase 2. Must have 1 month of on-time payments (or first payment recorded).
- **Status:** Required

**User sees:**
> A secured card works like a regular credit card, but you put down a deposit first. It reports to all three bureaus and builds payment history — the #1 factor in your score.

**Instructions:**
1. Choose your security deposit ($200 recommended minimum)
2. Select your card:
   - Best overall: Discover it® Secured
   - No-fee alternative: OpenSky® Secured Visa®
   - Upgrade option: Capital One Platinum Secured
3. Apply online (soft pull first with most issuers)
4. When approved, set up autopay for minimum payment
5. Use for 1-2 small purchases per month. Pay in full every month.

**Completion rule:** ⚠️ You cannot continue until you've had this card for 1 full billing cycle with an on-time payment.

**Sub-checklist:**
- [ ] Selected secured card
- [ ] Applied and approved
- [ ] Funded security deposit
- [ ] Set up autopay
- [ ] Made and paid for first purchase
- [ ] 1 full billing cycle with on-time payment ✅

---

### Task 9: Become an Authorized User ⚡ PARALLEL
- **Time:** 10 min
- **Gate:** Must complete Phase 2. Can run parallel with Task 8.
- **Status:** Optional but recommended

**User sees:**
> If a family member has an old credit card with perfect payment history, they can add you as an authorized user. Their positive history gets added to YOUR report.

**Instructions:**
1. Identify a trusted family member with:
   - Credit score 700+
   - Long credit history
   - Low utilization (under 30%)
   - Perfect payment history
2. Ask them to add you as authorized user
3. Confirm the card reports authorized users to credit bureaus (most do)
4. Wait 1-2 billing cycles for it to appear on your report

**Sub-checklist:**
- [ ] Identified potential authorized user candidates
- [ ] Asked family member to add you
- [ ] Confirmed card reports AU activity
- [ ] Waited 30 days for it to appear on report

---

### Task 10: Get a Credit-Builder Loan
- **Time:** 20 min
- **Gate:** Must complete Task 8.
- **Status:** Required (unless AU on a strong account)

**User sees:**
> A credit-builder loan works in reverse — you make payments first, get the money at the end. It adds an "installment loan" to your credit mix and reports on-time payments.

**Instructions:**
1. Recommended services:
   - Self Financial (easiest, fully online)
   - Credit Union credit-builder loans
   - Kikoff (alternative)
   - Atomic Investment (alternative)
2. Choose amount ($25-50/month for 12-24 months)
3. Set up autopay
4. Make every payment on time

**Sub-checklist:**
- [ ] Selected credit-builder service
- [ ] Signed up and funded account
- [ ] Set up autopay
- [ ] First payment made on time

**🏆 PHASE 3 COMPLETE:** "You've got active credit accounts reporting positive payment history. That's the engine driving your score up. Now we optimize."

---

### ⛔ CRITICAL GATE: Task 8 (1 billing cycle) must be done before Phase 4 unlocks.

## Phase 4: Optimize (Month 6-12)
**Theme:** Maximize your score.

---

### Task 11: Request Credit Limit Increases
- **Time:** 15 min per card
- **Gate:** 6+ months of on-time payments from Task 8
- **Status:** Required

**User sees:**
> A higher credit limit with the same spending = lower utilization = higher score. Ask for a raise.

**Instructions:**
1. Wait at least 6 months of on-time payments
2. Call or request online (online is fine)
3. Say: "I've been a responsible customer. Can you increase my credit limit?"
4. Accept if they ask about income — just tell the truth
5. Try one card at a time (in case there's a hard pull)

---

### Task 12: Apply for Better Credit Cards
- **Time:** 30 min
- **Gate:** Credit score must be 640+. Must complete Phase 3.
- **Status:** Required

**User sees:**
> Your credit is stronger now. Time to upgrade from a secured card to one that actually rewards you — based on how YOU spend.

**Instructions:**
1. Take the lifestyle quiz (see Credit Card Decision Tree)
2. Review personalized recommendations
3. Apply for your top pick
4. Set up autopay immediately
5. Close secured card and get deposit back (after 6-12 months)

---

### Task 13: Optimize Utilization
- **Time:** 15 min
- **Gate:** Must complete Task 12
- **Status:** Required

**User sees:**
> Utilization = how much of your available credit you're using. Under 30% is good. Under 10% is ideal. Here's how to get there.

**Instructions:**
1. Check current utilization: (total balances ÷ total limits) × 100
2. If over 30%, prioritize paying down highest-utilization cards
3. Consider: spreading charges across cards, requesting limit increases, paying twice per month
4. Set target: under 10% for maximum score boost

---

**Target Score Milestone:** If user started at 580 or below, they should now be 650-700+.

**🏆 PHASE 4 COMPLETE:** "Your credit isn't being repaired anymore. It's actively building. Well done."

## Phase 5: Maintain (Ongoing)
**Theme:** Keep it. Don't lose what you built.

---

### Task 14: Set Up Credit Monitoring
- **Time:** 15 min
- **Status:** Required

**Recommendations:**
- Free: Credit Karma, Credit Sesame (daily monitoring)
- Your bank may offer free FICO scores
- Pro tier: Automated monthly credit report pulls through our platform

### Task 15: Annual Credit Report Review
- **Time:** 30 min
- **Frequency:** Yearly
- **Status:** Required (recurring)

Set a calendar reminder for every January. Re-pull all three reports and look for:
- New errors
- Accounts that fell off (good — they were negative)
- Score trajectory

### Task 16: Score Goal Setting
- **Time:** 10 min
- **Status:** Optional
- **Frequency:** Quarterly

Set new goals:
- 700+ (good credit milestone)
- 740+ (great credit — best interest rates)
- 800+ (exceptional — all doors open)
